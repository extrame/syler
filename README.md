syler
=====

Syler Hua Wei Portal Middleware

Syler是一个支持华为Portal协议的开源网络管理中间件，支持协议包括

* 华为Portal 1.0 协议
* 华为Portal 2.0 协议

## 已测试设备

### 华为
S3700, AR 1220
### 华三
WX5000

## 下载
### 从源码安装

* go install github.com/extrame/syler
